Repeater display

The repeater is a full-screen, always-on notification board served by the device itself. Open it in a browser or an iframe on a dedicated display.

Open it

Browse to the device and add a query string:

  • http://<device>/?iphone=Brians-iPhone — focus one phone.
  • http://<device>/#repeater — show the phone picker.

URL parameters

  • iphone — the phone name to monitor (from the BLE/ANCS device name).
  • theme — a built-in preset: midnight, light, or terminal.
  • accent — override the accent color, e.g. accent=%23ff3b30.
  • css — URL of an external stylesheet loaded after the base theme for full control.

Theme it with CSS variables

The board is built on these tokens; override them via the css URL or the query params above:

  • --ancs-bg — background.
  • --ancs-fg — foreground/text color.
  • --ancs-accent — accent (connection dot, call ring).
  • --ancs-font — font family.
  • --ancs-radius — corner radius.
