ANCService

Turn a $5 ESP32 into an Apple Notification Center Service bridge. Plug the board into this computer with a USB cable and click Install — the installer also sets WiFi, shows logs, and links you into Home Assistant.

Your browser can't flash. Use desktop Chrome or Edge. Open this page over HTTPS to flash.

Normal install keeps saved WiFi and iPhone pairings. Use erase flash & install for a clean wipe (recovery, or re-pairing from scratch).

Trouble flashing? (“Failed to initialize”)
  1. Use desktop Chrome or Edge — not Safari, Firefox, or a phone.
  2. Use a data USB cable, and close anything else using the board (serial monitors, the ESPHome/Arduino IDE, other browser tabs).
  3. If the port picker shows more than one device, choose the one that matches your board’s USB chip — e.g. cu.SLAB_USBtoUART (CP2102) is usually more reliable than a generic usbserial adapter.
  4. Still failing? Your board’s auto-reset may not trigger from the browser. Hold BOOT, click Install, pick the port, and keep holding BOOT through “Connecting…” until it starts writing — then release.
  5. If even that fails: while it’s trying, do this on the board — hold BOOT, tap RESET/EN, release RESET, release BOOT. (Boards with native-USB chips like the ESP32-S3/C3 don’t need this.)
